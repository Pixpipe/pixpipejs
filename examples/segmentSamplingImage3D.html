Pixpipejs

This does the following :

  • open a local NIfTI/Minc2/MGH file, using pixpipe.FileToArrayBufferReader
  • redirect the file buffer into a pixpipe.Image3DGenericDecoder to extract the dataset and metadata
  • get the output as a generic pixpipe.Image3D object
  • perform a voxel sampling along the large diagonal
